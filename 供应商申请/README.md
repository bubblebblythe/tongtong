# OA供应商申请自动化

自动填写OA供应商申请表单，支持：
- **自动填写34/37字段**（92%自动化）
- 主表30个字段全自动
- 子表4个字段自动（联系人、手机、银行账号、户名）
- **手动字段**：性别、开户银行、附件上传

1. **OA账号**：H0029921
2. **Node.js** 和 **Playwright**

## 技术实现

### 核心机制

1. **Vue Model直接修改**
   - 通过 `__vue__` 访问表单Vue实例
   - 直接修改 `drawInitData.currentFormData` 的字段值
   - 调用 `$forceUpdate()` 触发UI更新

2. **子表动态行处理**
   - 自动点击"插入行"按钮（`.formson-list__button.cap-btn`）
   - 等待5秒让Vue生成 `records`
   - 通过 `record.lists.field0038` 访问子表字段

3. **枚举值精确匹配**
   - 从录制数据中提取真实的 `enumValue`
   - 例如：所属组织使用完整的"HongKong MagicEars Technology Limited (香港魔力耳朵科技有限公司)"

## 故障排查

### 问题1：性别字段不显示

**现象**：Vue model已设置 `field0040.value = "2"`，但UI不显示"女"

**原因**：
- 性别是动态下拉框，需要额外的DOM事件触发
- Vue model设置不足以让UI更新

**临时方案**：手动下拉选择"女"

### 问题2：开户银行无法自动填写

**现象**：开户银行是lookup字段，点击后弹出搜索窗口

**原因**：
- Lookup窗口在另一个iframe中
- 需要复杂的搜索、选择、确认流程

**临时方案**：手动lookup搜索"支付宝（中国）网络技术有限公司"

### 问题3："操作异常"报错

**原因**：字段值与枚举值不匹配

**解决**：
- 重新录制操作，提取真实的 `enumValue`
- 确保使用完整的字段值（如组织名称包含中英文）
