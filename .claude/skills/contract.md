# OA合同申请自动化

自动化填写豌豆思维OA合同申请表单，支持人民币和外币合同。

## 功能

- 自动识别币种（人民币/美元/港币等）
- 智能选择我方签约主体（人民币→广州豌豆思维，外币→香港魔力耳朵）
- 自动填写票据类型、开票形式、发票税率（人民币对私6%，外币0%）
- 自动搜索对方签约主体（人民币对私→广西信华+中国银行，外币→合同乙方）
- 自动上传合同和ROI附件
- 保存到OA待发箱

## 使用方法

```
/contract contract="合同路径" roi="ROI路径"
```

### 示例

```
/contract contract="C:\交接-悦彤\社群\依娃Yvonne\合同\依娃Yvonne&广西信华服务协议附件.docx"
```

或带ROI：

```
/contract contract="C:\交接-悦彤\社群\依娃Yvonne\合同\依娃Yvonne&广西信华服务协议附件.docx" roi="C:\交接-悦彤\社群\依娃Yvonne\外渠ROI测算模板.xlsx"
```

## 参数

- `contract`（必填）：合同Word文档的完整路径
- `roi`（可选）：ROI Excel文档的完整路径

## 运行逻辑

执行以下命令：

```bash
cd "C:/Users/liangyuetong/Desktop/合同OA自动化"
node run_real.mjs --contract "<合同路径>" [--roi "<ROI路径>"]
```

## 注意事项

1. **合同路径必须使用完整路径**，不支持相对路径
2. 路径中如有空格，会自动处理
3. 脚本会自动打开Edge浏览器，登录OA，填写表单
4. 填写完成后浏览器会保持打开5分钟供检查
5. 检查待发箱中的合同信息无误后，手动点击"提交"

## 支持的币种

- **人民币对私**：广州豌豆思维 + 广西信华(中国银行) + 增值税电子普票 + 先款后票 + 6%税率
- **人民币对公**：广州豌豆思维 + 合同乙方 + 增值税电子普票 + 先票后款 + 1%税率
- **外币（美元/港币等）**：香港魔力耳朵 + 合同乙方 + invoice + 先票后款 + 0%税率

## 关键字段自动填写

- ✅ 我方签约主体（根据币种）
- ✅ 对方签约主体（自动lookup）
- ✅ 银行账号（智能匹配）
- ✅ 币种和汇率
- ✅ 票据类型（根据币种）
- ✅ 开票形式（根据币种和对方类型）
- ✅ 发票税率（根据币种和对方类型）
- ✅ 合同名称、渠道名称
- ✅ 合同日期
- ✅ 合同附件上传

## 离职交接说明

**关键文件**：
- `run_real.mjs` - 主脚本
- `REAL_VALUES` - 需要根据每个合同手动更新的字段（合同名称、渠道名称、日期等）
- `乙方账号映射表.json` - 渠道账号信息

**下次使用前**：
1. 打开 `run_real.mjs`
2. 找到 `REAL_VALUES` 对象（第28-77行左右）
3. 更新以下字段：
   - `contractName`: 合同名称
   - `channelName`: 渠道名称
   - `startDate` / `endDate`: 合同起止日期
   - `cooperationContent`: 主要合作内容
   - `settlementStandard`: 结算标准
   - `paymentPlan`: 付款计划

**或者**：让AI帮你从合同里提取这些信息并自动更新代码。

## 故障排查

1. **浏览器未打开**：检查Edge浏览器是否已安装
2. **登录失败**：检查OA账号密码配置
3. **字段未填写**：检查REAL_VALUES是否正确配置
4. **保存到草稿箱而不是待发箱**：脚本已修复，优先点击"保存待发"
5. **税率/主体选择错误**：检查币种识别逻辑

## 技术细节

- 使用Playwright自动化浏览器
- 使用真实录制的enumValue确保下拉框正确填写
- 支持人民币对私自动搜索广西信华并匹配中国银行账号
- DOM补填和fillValues双重保障关键字段填写
