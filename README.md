# OA合同申请自动化系统

## 简介

自动化填写豌豆思维OA合同申请表单的工具，支持人民币和外币合同。

**一键完成**：自动识别币种 → 选择主体 → 填写税率 → 搜索对方 → 上传附件 → 保存待发

## 快速使用

## 常见问题

### Q1: 字段填错了怎么办？

**A**: 脚本运行后会保持浏览器打开5分钟，在待发箱中检查字段，如有错误可以手动修改后再提交。

### Q2: 如何修改默认配置？

**A**: 编辑 `run_real.mjs`，找到 `REAL_VALUES` 对象（第28-77行），修改对应字段的值。

### Q3: 支持哪些币种？

**A**: 支持人民币、美元、港币。其他币种可能需要调整脚本。

### Q4: 为什么保存到草稿箱而不是待发箱？

**A**: 已修复，脚本现在优先点击"保存待发"按钮。如果仍保存到草稿箱，请联系技术支持。

### Q5: 发票税率显示不对

**A**: 检查币种识别是否正确。人民币对私应该是6%，人民币对公是1%，外币是0%。

### Q6: 对方主体没有自动填写

**A**: 人民币对私会自动搜索"广西信华"并匹配中国银行账号。其他情况使用合同乙方名称，需要在OA系统中已存在该供应商。

- **表单填写**：结合真实enumValue和DOM操作确保准确性
